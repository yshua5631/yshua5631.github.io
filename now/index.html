Now

这是一个 /now 页面，记录我当前的关注方向，区别于长期身份（About）和某次思考（文章）。每季度至少更新一次。

当前聚焦

AI 技术的工程化落地。 模型能力已经不再是瓶颈，真正的挑战在于如何把 LLM 可靠地嵌入到生产系统里——延迟、成本、可观测性、失败兜底、人机边界，这些问题背后都是经典的分布式系统和架构问题，只是换了一层皮。

我的视角是后端工程师的视角：不关心炼丹，关心怎么把炼好的丹装进产线。

正在探索

  • Agent 系统的工程化：Claude Code / Cursor 这类编码智能体的架构设计，上下文窗口管理、工具调用编排、长任务的状态恢复
  • MCP（Model Context Protocol）：作为 AI 应用的”USB 接口”，如何构建可复用的 Server 生态，以及企业内部怎么做 MCP 治理
  • RAG 架构的演进：从朴素向量检索到 GraphRAG、Agentic RAG，检索召回率和答案质量之间的 trade-off
  • Context Engineering：Prompt 工程之后更大的话题——如何在有限的上下文窗口里精准投喂信息，缓存策略、压缩策略、多轮对话的状态管理
  • AI 产品的可观测性：LLM 应用的 tracing、eval、回归测试体系，以及怎么度量”模型质量”这种主观指标

职业与成长

  • 探索职业生涯的新方向。软件工程师这个身份之外，还有哪些可能性？AI 浪潮让很多旧经验贬值、也让很多新角色冒出来，不想被惯性推着走，想主动选
  • 坚持长期主义。短期的噪音太多——风口、焦虑、KPI——真正重要的东西需要用年为单位去积累。不赚快钱，不做一次性投入，只做能复利的事
  • 终身成长。不把学习当成阶段性任务，而是默认状态。读书、写作、折腾新技术、跟更优秀的人对话——这些不是”业余爱好”，而是正事
